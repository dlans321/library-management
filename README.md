# library-management
Console 기반의 도서관 좌석 관리 프로그램입니다. 좌석 예약/연장/반납 등의 사용자 기능과 열람실 추가/활성·비활성화 등의 관리자 기능을 구현했습니다.

## Usage
프로젝트를 빌드하고 실행 파일을 실행합니다.
